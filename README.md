# photo-extractor
- target 경로, dest 경로를 입력하면 내부배열의 데이터를 포함하고 있는 파일들을 target에서 찾아, dest로 복사 해주는 python script

##### 사용 유틸
- os:파일과 디렉토리를 다루는 데 사용
- shutil: 파일을 복사하는 데 사용
